# Task Flow App

A simple Task Flow App built using React and Vite.

## Features

-Add Tasks
-Delete Tasks
-Mark Tasks as Completed
-Search Tasks
-Filter Tasks
-Task Categories
-Dark Mode
-Stats Page
-Responsive Design
-Local Storage Support

## Technologies Used

-React
-Vite
-Tailwind CSS
-React Router DOM

## Pages

-Homepage
-Stats Page
-Settings page

## Responsive

This app works on:
-Mobile
-Tablet
-Laptop
